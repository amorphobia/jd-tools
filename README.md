# 星空键道工具

## 键道捕获器

![捕获器界面](./pic/catcher-snapshot.png)

包含一个主要脚本和两个依赖的动态库，需要放到同一个目录下：

```text
catcher.ps1
git2-e632535.dll
LibGit2Sharp.dll
```

使用时，右键点击 `catcher.ps1` 选择「使用 PowerShell 运行」，或在 PowerShell 命令行直接运行即可。
